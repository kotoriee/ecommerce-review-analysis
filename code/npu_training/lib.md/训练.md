参数说明

新建一个计算调试任务，填入以下参数来配置你的调试环境

算力集群 选择智算集群
计算资源 选择NPU
访问Internet 选择是否需要在环境内访问网络，部分NPU资源无法提供访问外部网络
任务名称 只能包含小写字母，数字，以及下划线与连接号，不能以下划线结尾，最多36个字符
任务描述 不超过255个字符，可空白
代码分支 此选项将在决定后台导入调试环境的代码仓分支，默认为master分支
镜像 只能选择平台提供 预设镜像，不可提交自定义镜像或使用外部镜像
模型 若不使用模型，可空白
后台将自动导入模型到调试环境
模型可多选，在弹窗中选中多个模型即可，总大小不能超过70G
你也可以在调试代码里下载外部模型
数据集 若不使用数据集，可空白
后台将自动导入数据集到调试环境
数据集可多选，在弹窗中选中多个数据集即可，总大小不能超过20G
你也可以在调试代码里下载外部数据集
资源规格 提供Ascend 910算力卡
Modelart 控制台
与GPU调试任务不同，NPU调试提供华为 Modelarts 控制台

提示

请注意，NPU调试任务无法使用 pip 或 conda 自行安装第三方库
traindetail
拉取代码仓
NPU调试任务需要手动下载代码仓与数据集，你不需要改变NPU调试环境的工作目录，所有文件都应放在默认路径下

在控制台中，点击左侧搜索框上方最右边的Git图标，输入代码仓地址
你也可以在 Terminal 中使用 git clone 代码仓地址 来克隆代码仓
你也可以使用 c2net 库来获取代码仓路径，详情请参考 环境初始化

使用 c2net 库在环境中获取数据集路径，详情请参考 环境初始化
